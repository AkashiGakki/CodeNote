# Function

## 三、理解函数

> 把 `JavaScript` 作为函数式语言来理解。

### 3.1 函数是第一类对象

> `JavaScript` 中最关键的概念是：函数是第一类对象（`first-class objects`）或者说函数是一等公民。

函数和普通的 `JavaScript` 数据类型一样，可以：

- 通过字面量创建

```js
function func () {}
console.log(typeof func)
```

- 赋值给变量、数组项或其他对象的属性

```js
const func = function() {}

let arr = [];
arr.push(function () {});

arr.func = function() {}
```

- 作为函数的参数来传递

```js
const funcA = function () {
  console.log(1);
}
const funcB = function (func) {
  func();
}

funcB(funcA);
```

- 作为函数的返回值

```js
const funcC = function () {
  console.log(2);
}
const funcD = function () {
  return funcC();
}

funcD();
```

- 具有动态创建和分配的属性

```js
const func = function () {}
func.user = "akashi";
console.log(func.name);
```
